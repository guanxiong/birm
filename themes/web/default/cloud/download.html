微新星产品介绍

公众平台自助引擎，简称微新星，微新星是一款免费开源的微信公众平台管理系统。

注：使用微新星必须要有自己的服务器、虚拟空间或者自己的BAE，且必须要支持PHP+MYSQL，上传微新星程序后按系统指引安装即可。

微新星 v0.51 以后将采用自动升级的方式提供补丁和升级版本。为保证程序安全性和第一时间获取最新功能，请及时按照提示进行自动更新。

微新星安装方法

下载 微新星 v0.51 将微新星独立安装文件 install.php 上传到服务器或者虚拟空间后访问 {安装URL}/install.php 按照系统指引安装即可，系统将会在过程中自动下载最新的程序文件。

微新星版本说明

微新星从 0.51 开始将提供两条产品线，公众版和商业版。这两个版本的核心是一致的，商业版会提供一些附加的商业模块，并且附带技术支持和产品授权(允许用作商业运营)。购买商业版，请联系微新星团队。
